Coordinate-Ascent
=================
Implementation of a basic Coordinate Ascent (same thing as Coordinate Descent, just 'reversed'), as well as the Lasso implementation.

## Modules
This repository has basically 4 modules:

* Gradient Descent
* File to test everything

## Requirements

* Python 3.4
* Numpy >= 1.9
* Matplotlib >= 1.4.0
